Google permissions used
Calendar
Read-only. Used to display upcoming events in the check-in screen and provide schedule context to the AI assistant. The app never creates or modifies calendar events.
Tasks
Read and write. Used to display existing tasks, create new tasks from brain dump sessions, update due dates, and mark tasks complete during check-in.
Docs
Read and write. Used to append ideas to a single Google Doc that you create yourself and specify in Settings. No other documents are accessed.
